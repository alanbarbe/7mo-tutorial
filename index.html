Instalación de Sistema Operativo

1. Preparación

Verifica las especificaciones mínimas de hardware requeridas para la versión de Windows que deseas instalar. Por ejemplo, para Windows 11, necesitarás un procesador de 1GHz o más rápido con 2 o más núcleos, 4GB de RAM, 64GB de almacenamiento y una tarjeta gráfica compatible.

Conecta un teclado y mouse USB a tu notebook si no tiene este tipo de dispositivos integrados. Asegúrate de que funcionen correctamente.

Descarga la imagen ISO de Windows desde la página oficial de Microsoft. Hay diferentes ediciones (Home, Pro, etc.) así que elige la que necesites.

Consigue una unidad USB vacía con al menos 8GB de capacidad de almacenamiento. Es recomendable usar una unidad nueva o formatearla completamente.

Descarga e instala un programa para crear unidades USB booteable como Rufus. Este es gratuito y compatible con Windows.

Enlaces oficiales para descargar:

  • Descargar ISO
  • Descargar Rufus

2. Creación de un USB Booteable

Abre el programa Rufus una vez instalado.

En la opción "Dispositivo" selecciona tu unidad USB de al menos 8GB. Haz clic en "Seleccionar" y elige la imagen ISO de Windows que descargaste previamente.

En "Esquema de partición" es recomendable elegir "Tabla de Partición GPT" para unidades de 4GB o más grandes.

Bajo "Sistema de archivos", selecciona "NTFS" como formato para la unidad. Las demás opciones puedes dejarlas con la configuración por defecto.

Una vez revisada la configuración, haz clic en "Comenzar" para iniciar el proceso de creación de la unidad booteable.

Rufus mostrará una advertencia, haz clic en "Aceptar" para proceder. La copia de archivos a la unidad USB comenzará y puede tardar varios minutos dependiendo de tu equipo.

Una vez finalizado, Rufus mostrará un mensaje de "¡Listo!". Ahora tienes una unidad USB booteable con la imagen de instalación de Windows lista para usar.

3. Configuración del BIOS/UEFI

Reinicia tu notebook.

Inmediatamente después de encender, presiona repetidamente la tecla de acceso al BIOS/UEFI. Las teclas más comunes son:

  • F2 (para la mayoría de las marcas)
  • Del (para algunas marcas como Asus)
  • F10 (para HP)
  • Esc (para algunas Lenovo)

Dentro del BIOS/UEFI, navega hasta el menú de arranque y cambia el orden de arranque para que la USB sea la primera opción.

Habilita el modo Legacy o UEFI según sea necesario.

Guarda los cambios y reinicia tu notebook para arrancar desde la USB.

4. Instalación de Windows

Paso 1: Arranque desde USB

  • Si configuraste correctamente el BIOS/UEFI, tu notebook arrancará desde el USB.
  • Verás un mensaje como "Press any key to boot from USB".

Paso 2: Inicio de la instalación

  • Aparecerá la pantalla de carga de Windows.
  • Selecciona tu idioma, formato de hora y método de entrada de teclado.
  • Haz clic en "Siguiente" y luego en "Instalar ahora".

Paso 3: Clave de producto

  • Si tienes una clave, ingrésala. Si no, selecciona "No tengo clave de producto".

Paso 4: Selección de edición de Windows

  • Elige la edición que corresponda a tu clave (si la ingresaste) o la que desees instalar.

Paso 5: Aceptar términos de licencia

  • Lee y acepta los términos de licencia.

Paso 6: Tipo de instalación

  • Selecciona "Personalizada: Instalar solo Windows (avanzado)".

Paso 7: Selección y partición del disco

  • Selecciona el disco donde instalarás Windows.
  • Si es necesario, crea nuevas particiones o formatea las existentes.
  • Selecciona la partición donde instalaremos Windows y haz clic en "Siguiente".

Paso 8: Instalación

  • Windows comenzará a copiar archivos e instalarse. Esto puede llevar varios minutos.
  • Tu notebook se reiniciará varias veces durante este proceso.

Paso 9: Configuración inicial

  • Elige tu región y disposición de teclado.
  • Configura tu cuenta de usuario (nombre y contraseña).
  • Decide sobre las opciones de privacidad (ubicación, diagnóstico, etc.).

Paso 10: Finalización

  • Windows terminará de configurarse y llegarás al escritorio.

Paso 11: Instalación de controladores

  • Ve a la página del fabricante de tu notebook y descarga los controladores más recientes.
  • Instala los controladores uno por uno, reiniciando cuando sea necesario.

5. Post-Instalación y Configuración Adicional

Paso 1: Actualizar Windows

  1. Abre el menú Inicio y ve a Configuración (icono de engranaje).
  2. Selecciona "Actualización y seguridad".
  3. Haz clic en "Windows Update" en el panel izquierdo.
  4. Haz clic en "Buscar actualizaciones".
  5. Si hay actualizaciones disponibles, Windows las descargará e instalará automáticamente.
  6. Reinicia tu notebook cuando se te solicite para completar la instalación de las actualizaciones.

Paso 2: Instalar controladores

  1. Método automático (Windows Update):
    • Windows generalmente instala controladores básicos automáticamente.
    • Para buscar controladores adicionales, ve a Configuración > Actualización y seguridad > Windows Update > Opciones avanzadas > Opciones opcionales.
    • Haz clic en "Ver actualizaciones opcionales" y busca actualizaciones de controladores.
  2. Método manual (herramientas del fabricante):
    • Visita el sitio web oficial del fabricante de tu notebook.
    • Busca la sección de soporte o descargas.
    • Introduce el modelo exacto de tu notebook.
    • Descarga e instala los controladores más recientes, especialmente para:
      • Tarjeta gráfica
      • Chipset
      • Red (Wi-Fi y Ethernet)
      • Audio
    • Reinicia tu notebook después de instalar los controladores importantes.

Paso 3: Instalar software esencial

  1. Navegador web:
    • Descarga e instala tu navegador preferido (Chrome, OperaGX, etc.)
  2. Suite de oficina:
    • Instala Microsoft Office o una alternativa gratuita como LibreOffice.
  3. Antivirus:
    • Windows incluye Windows Defender, pero si prefieres otra opción, instálala ahora.
  4. Otros programas útiles:
    • Lector de PDF (Adobe Acrobat Reader o alternativas)
    • Compresor de archivos (7-Zip, WinRAR)
    • Reproductor multimedia (VLC)

Paso 4: Optimizar el rendimiento

  1. Ajustar el plan de energía:
    • Ve a Panel de control > Sistema y seguridad > Opciones de energía.
    • Selecciona "Alto rendimiento" o crea un plan personalizado.
  2. Desactivar efectos visuales innecesarios:
    • Haz clic derecho en "Este equipo" > Propiedades > Configuración avanzada del sistema > Rendimiento > Configuración.
    • Selecciona "Ajustar para obtener el mejor rendimiento" o personaliza según tus preferencias.
  3. Desactivar programas de inicio:
    • Abre el Administrador de tareas (Ctrl + Shift + Esc).
    • Ve a la pestaña "Inicio".
    • Desactiva los programas que no necesites que se inicien con Windows.
  4. Liberar espacio en disco:
    • Usa la herramienta Liberador de espacio en disco de Windows.
    • Ve a Este equipo > haz clic derecho en el disco C: > Propiedades > Liberador de espacio en disco.

Paso 5: Eliminar bloatware

  1. Desinstalar aplicaciones preinstaladas no deseadas:
    • Ve a Configuración > Aplicaciones > Aplicaciones y características.
    • Revisa la lista de aplicaciones instaladas.
    • Selecciona y desinstala las que no necesites.
  2. Usar herramientas de limpieza (opcional):
    • Puedes usar programas como CCleaner o PC Decrapifier para eliminar bloatware de forma más exhaustiva.
    • Ten cuidado al usar estas herramientas y asegúrate de no eliminar aplicaciones importantes.
  3. Reinstalar aplicaciones útiles del fabricante:
    • Algunas aplicaciones preinstaladas pueden ser útiles (por ejemplo, para controlar el hardware específico de tu notebook).
    • Si las eliminaste por error, puedes reinstalarlas desde el sitio web del fabricante.
